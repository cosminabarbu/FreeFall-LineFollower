# FreeFall-LineFollower
10h hackathon as part of the final project for Introduction To Robotics 3rd year course.

# Components used 
* Arduino uno
* Small breadboard
* L239D motor driver
* 2 DC motors
* 2 wheels
* A power source AKA a LiPo battery in our case
* QTR-8A Reflectance Sensor
* Ball caster
* Zip-ties, all types of wires, screws
